Simon Game
A web-based recreation of the classic Simon Memory Game, built with HTML, CSS, JavaScript, and jQuery.
Test your memory and see how many color sequences you can remember and repeat correctly!

About the Game
The Simon Game is a simple and fun pattern memory challenge:
- The game shows you a random color by flashing one of the four colored buttons.
- You must click the same color
- On the next level, Simon adds one more color to the sequence and you have to remember and repeat the entire pattern in the correct order.
- If you click the wrong color, the game ends and you start over.

Built With
HTML5 - Structure and layout
CSS - Styling and animations
JavaScript (ES6) - Game logic and interactivity
jQuery - DOM manipulation and event handling


How to Play
- Download or clone this repository:
git clone https://github.com/YourUsername/simon-game.git
- Open the project folder and run index.html 
- Press any key to start the game.
- Watch the color sequence, then click the colored buttons in the same order
- The sequence grows longer each round, try to remember them all!
